net.virtualvoid.codec.CryptCodecs

EncryptCBC

case class EncryptCBC (cipherName: String, key: SecretKeySpec) extends CodecBase[Bytes, (Bytes, Bytes)] with Product with Serializable

Encrypts an array of bytes using a javax.crypto.Cipher with the specified key. The result is a tuple (<initialization-vector>, <encrypted-data>). The initialization vector is generated randomly so encoding results in different output each time it is called.

cipherName

The name of a javax.crypto.Cipher

key

The secret key for this cipher
